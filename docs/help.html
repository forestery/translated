- How to use
  • You can press "Ctrl+Shift+F" to open this extension.
  • Select a word on web ,then just press "Ctrl+Shift+F" to translate the word between English and Chinese.
  • Type a word in the input field and press Enter key.
  • In the new word list field, click the word to translate, click remove link when you memorize it.

- About Translated
  • Version: v0.1
  • A chrome exetension to help translate between English and Chinese.It aim to provide friendly UI and try to keep dead easy.
  • The translate engin is powered by 微软必应词典.
